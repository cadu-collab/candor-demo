## Demo Flow (8–10 min)

| # | Section | Duration | What to do |

## Session Demo Notes

- Click **▶ Play Session** to start both conversations simultaneously
- Both columns run in parallel — the contrast is visible in real time
- Annotations appear after each Alex message — these explain what is happening behaviorally
- Outcomes reveal at the end of each column
- Click **Reset** to replay (e.g., for a second audience member or a VC who wants to see it again)
- Total animation time: approximately 3 minutes

## Running Locally

Open `index.html` directly in Chrome or Firefox. No server, no install required.

For a live presentation, use the GitHub Pages URL rather than opening from the filesystem — it avoids any local path issues and works from any device on the network.

---

## GitHub Pages Setup

1. Create a new public repository named `candor-demo`
2. Push `index.html` and `README.md` to the `main` branch
3. Go to Settings → Pages → Source: Deploy from branch → main → / (root)
4. GitHub will provide the live URL (takes ~60 seconds to activate)
5. Share the URL with co-founders for review before the event

Only push `index.html` and `README.md`. Do not push pitch decks, financial models, or strategy documents to a public repository.
